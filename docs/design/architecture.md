# 系统架构设计

## 概述

Sentinel-X 是一个分布式智能运维系统，旨在提供高可用、可扩展的监控和自动化运维能力。

## 核心组件

1.  **API Server**: 提供 RESTful API 接口，处理用户请求。
2.  **Scheduler**: 任务调度器，负责分发运维任务。
3.  **GoAgent**: 核心代理库，封装了与底层系统交互的逻辑（作为 Monorepo 的一部分进行管理）。

## 目录结构说明

*   `cmd/`: 应用程序入口。
*   `internal/`: 私有应用代码。
*   `pkg/`: 可被外部引用的库代码。
*   `staging/`: 核心依赖库源码 (Source of Truth)。
*   `vendor/`: 第三方依赖副本。
